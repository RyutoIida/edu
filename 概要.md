# 調整凍結
```bash
python main.py --input ./data/input_data1.json --mode tune --lock-out ./lock/lock_ds1 --template ./templates/spec.md.j2 --output-md ./output/ds1/spec.md --model gpt-4o --temperature 0.2 --profile ./profile.json

```
- --input: 入力データファイルの指定
- --mode:
  - turn: lockを作成
- --lock-out: lockファイル名の指定
- --template: 仕様テンプレートの指定
- --out: 出力ファイル名の指定
  
# 無調整で評価
```bash
python main.py --input ./data/input_data2.json --mode eval --lock-in ./lock/lock_ds1 --output-md ./output/ds2_from_ds1lock/spec.md
```
- --input: 入力データファイルの指定
- --mode:
  - eval: lockを読み込み、無調整適用
- --lock-in: 読み込むlockファイルの指定
- --template: 仕様テンプレートの指定
- --out: 出力ファイル名の指定
